# data-schema.md · 报告数据 JSON 字段说明

> 这是传入 `generate_report.py` 的数据 JSON 完整字段规范。  
> 标注 `[必填]` 的字段缺失会导致报告渲染失败；`[可选]` 字段缺失仅跳过对应模块。

---

## 顶层字段

```json
{
  "keyword": "穿搭痛点",          // [必填] 调研关键词，用于标题和文件名
  "date": "2026-04-03",           // [必填] 调研日期 YYYY-MM-DD
  "note_count": "1200万+",        // [必填] 笔记总量（可估算，标注AI估算）
  "keyword_count": "12",          // [可选] 相关关键词数量
  "max_likes": "10万",            // [必填] 最高单篇点赞数（用于 Header）
  "heat_level": "极高",           // [必填] 热度级别（极高/高/中/低）
  "overview_cards": [...],        // [必填] 数据概览卡片
  "keyword_heat": [...],          // [可选] 关键词热度列表
  "background": {...},            // [可选] 赛道背景模块
  "top_notes": [...],             // [必填] TOP 代表性笔记
  "creators": [...],              // [必填] 活跃创作者画像
  "comments": [...],              // [可选] 用户真实声音
  "needs_wants": [...],           // [可选] 用户需求列表（字符串数组）
  "needs_pains": [...],           // [可选] 用户痛点列表（字符串数组）
  "insights": [...]               // [可选] 核心洞察列表
}
```

---

## overview_cards（数据概览卡片）

```json
"overview_cards": [
  {
    "label": "穿搭赛道笔记总量",    // [必填] 指标名称
    "value": "1200万+",             // [必填] 指标数值
    "desc": "「穿搭」是小红书历史体量最大的内容品类" // [必填] 补充说明
  }
]
```

建议 4-6 张，内容参考：
- 笔记总量
- 单篇最高点赞
- 核心赛道增速（可AI估算，需标注）
- 核心用户群
- 内容转化率特点
- 发布黄金时段

## keyword_heat（关键词热度）

```json
"keyword_heat": [
  {
    "name": "身材焦虑类（梨形/矮个子/微胖）",  // [必填] 关键词/话题名
    "stars": 5,                                   // [必填] 热度星级 1-5
    "percent": 98                                 // [必填] 热度百分比 0-100（用于进度条）
  }
]
```

建议 6-10 条，按热度从高到低排列。

---

## background（赛道背景）

```json
"background": {
  "title": "穿搭赛道背景（2025-2026年趋势速览）",  // [必填] 标题
  "cards": [                                          // [必填] 信息卡片列表
    {
      "title": "📈 2025-2026爆发风格",  // [必填] 卡片标题（建议加emoji）
      "items": [                          // [必填] 要点列表（字符串数组）
        "动画/IP角色配色穿搭：2026年最强新兴爆款路径",
        "新中式/国风：增速最猛，年增26倍"
      ]
    }
  ]
}
```

建议 3-4 张背景卡片，主题参考：
- 爆发风格/趋势
- 核心用户痛点
- 爆款笔记三要素
- 关键数据洞察

## top_notes（TOP 代表性笔记）

```json
"top_notes": [
  {
    "title": "跟着动画角色配色学春日穿搭就会得到…",  // [必填] 笔记标题
    "url": "https://www.xiaohongshu.com/explore/698ae59e000000002801c8ad",  // [必填] 笔记真实URL（必须验证可访问）
    "author": "白昼小熊",                              // [必填] 作者昵称
    "author_url": "https://www.xiaohongshu.com/user/profile/5a8cf39111be10466d285d6b",  // [必填] 作者主页URL
    "desc": "以动画角色配色为切入点...",               // [必填] 2-3句爆款原因分析（AI分析）
    "tags": [                                           // [可选] 标签（最多3个）
      {"text": "🏆 当月爆款NO.1", "color": "red"},    // color: red/blue/green
      {"text": "创意跨界", "color": "red"}
    ],
    "likes": "10万",      // [必填] 点赞数（从页面真实采集）
    "bookmarks": "2.5万", // [必填] 收藏数（从页面真实采集）
    "comments": "5479"    // [必填] 评论数（从页面真实采集）
  }
]
```

> ⚠️ `url` 和 `author_url` **必须通过浏览器实际访问验证**，不能构造。  
> 建议 3-5 条笔记，按点赞数从高到低排列。

## creators（活跃创作者画像）

```json
"creators": [
  {
    "emoji": "🌸",                   // [必填] 类型图标
    "type": "头部潮流博主",           // [必填] 类型名称
    "names": [                        // [必填] 博主列表（至少2个）
      {
        "name": "白昼小熊",           // [必填] 博主昵称
        "url": "https://www.xiaohongshu.com/user/profile/5a8cf39111be10466d285d6b",  // [必填] 主页URL
        "fans": "368.2万"             // [必填] 粉丝数（从主页实时采集）
      }
    ],
    "desc": "白昼小熊 368万粉，动画配色×穿搭跨界...",  // [必填] 该类型博主特征描述
    "topics": [                       // [可选] 典型选题（从博主真实笔记列表提取）
      "跟着动画角色配色学穿搭",
      "解决被你忽略的穿衣习惯（系列）"
    ]
  }
]
```

> ⚠️ `names` 中每个博主的粉丝数 **必须通过访问主页实时核实**，不能估算。  
> 每种类型至少 2 个博主，推荐 2-4 个。

## comments（用户真实声音）

```json
"comments": [
  {
    "text": "小熊姐是色彩之神！！！真的学到了",  // [必填] 评论内容
    "source": "跟着动画角色配色学春日穿搭",       // [必填] 来源笔记标题
    "note": "（7818赞，真实评论）"               // [可选] 备注说明
  }
]
```

建议 4-8 条，混合真实采集评论和典型评论类型描述。

## needs_wants / needs_pains（需求与痛点）

```json
"needs_wants": [
  "针对自己身材的专属穿搭公式",
  "「真实场景可穿」的日常穿搭"
],
"needs_pains": [
  "身材不标准，市面上大多数版型都不适合",
  "「穿衣习惯」错误但自己完全不知道"
]
```

每项 4-8 条字符串，AI 综合分析生成。

```json
    "desc": "白昼小熊「跟着动画角色配色学穿搭」拿到10万赞..."   // [必填] 1-2句详细说明
  }
]
```

建议 3-5 条，每条包含具体数据支撑，可执行性强。
